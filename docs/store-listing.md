# ストア掲載文草案

最終更新日: `2026-03-16`

## 拡張機能名

X Popular Link Filter

## 短い説明文

Xホームタイムラインから人気のある外部リンク付き投稿を抽出し、ポップアップで一覧表示します。

## 詳細説明文

X Popular Link Filter は、現在開いている X のホームタイムラインを解析し、外部リンクを含む投稿を収集します。

収集した投稿のうち、いいね数・リポスト数・返信数のいずれかが一定の閾値を超えた投稿だけを抽出し、拡張機能のポップアップに一覧表示します。

主な特徴:

- Xホームタイムライン上の投稿を対象にする
- 外部リンク付き投稿だけを抽出する
- いいね / リポスト / 返信数で人気投稿を絞り込む
- 結果をポップアップで確認できる
- 投稿URLと外部リンクURLの両方を確認できる

## 審査担当者向けメモ

Xにログインした状態で `https://x.com/home` を開き、拡張機能アイコンをクリックして `Collect Now` を押してください。

拡張機能は、表示中のタイムライン投稿と自動スクロールで追加読み込みされた投稿を走査し、外部リンク付き投稿のうち固定閾値を超えた投稿のみをポップアップに表示します。

## プライバシー説明の短文

この拡張機能は、Xホームタイムライン上の投稿を読み取り、外部リンク付き投稿の抽出と人気判定を行います。取得した投稿情報は外部サーバーへ送信しません。
